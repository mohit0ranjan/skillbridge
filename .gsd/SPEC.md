# SPEC.md — Project Specification

> **Status**: `FINALIZED`

## Vision
Transform the SkillBridge frontend from a template-like site into a premium, human-crafted, and trustworthy SaaS product through a deep UI/UX audit and comprehensive design system overhaul.

## Goals
1. Establish a strict Design System (typography, minimal palette, 8px spacing grid).
2. Standardize all components (cards, buttons, inputs) with consistent heights, visual hierarchies, and modern styling (soft shadows, hover lifts).
3. Redesign all email templates (welcome, payment, certificates) to emulate top-tier SaaS aesthetics.
4. Fix responsive layouts across mobile, tablet, and desktop views while running a cleanup of unused styles.

## Non-Goals (Out of Scope)
- Core functionality features outside the scope of UI scaling and presentation.
- Expanding into completely new pages not existing currently.

## Users
Students seeking internships, looking for a trustworthy, premium platform that instills confidence similar to top-tier enterprise software products.

## Constraints
- Must strictly use the provided color palette and font combo.
- Must ensure clean separation between UI components without heavy customization that creates maintenance debt.

## Success Criteria
- [ ] Typography and colours consistently applied across all pages.
- [ ] No visual inconsistencies (padding, spacing, margins) on the homepage, dashboard, and public pages.
- [ ] All cards and inputs standardized without basic generic UI aesthetics.
- [ ] Email templates tested and fully redesigned.
